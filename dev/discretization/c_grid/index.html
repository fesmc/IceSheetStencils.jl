C-grid (SSA)

This page documents the only stencil currently shipped: the SSA momentum balance discretised on an Arakawa C-grid with periodic boundary conditions. Every symbolic expression below is rendered directly from the package via Symbolics.jl + Latexify.jl, so the page is guaranteed to track the source.

Variable layout

For an Nx × Ny cell-centred grid with spacing (dx, dy):

QuantityLocationArray indexPhysical position
$H,\, \eta,\, \beta,\, s$cell centre[i, j]$\bigl((i-\tfrac12)\,dx,\; (j-\tfrac12)\,dy\bigr)$
$u$east face[i, j]$\bigl(i\,dx,\; (j-\tfrac12)\,dy\bigr) = u_{i+1/2,\,j}$
$v$north face[i, j]$\bigl((i-\tfrac12)\,dx,\; j\,dy\bigr) = v_{i,\,j+1/2}$

The state vector is [u_flat; v_flat] with column-major flattening of each Nx × Ny array. Periodic BCs wrap $i \rightarrow i \pm Nx$, $j \rightarrow j \pm Ny$.

Local-stencil notation

Every symbolic expression on this page uses the local labelling carried by StencilSpec. For the x-momentum equation centred on a u-point at $(i+\tfrac12,\, j)$:

Local nameField, positionGlobal index
u_m1_0, u_0_0, u_p1_0$u$ at cells west / centre / east of the u-point in $x$[i-1, j], [i, j], [i+1, j]
u_0_m1, u_0_p1$u$ at cells south / north of the u-point in $y$[i, j-1], [i, j+1]
v_w_s, v_e_s, v_w_n, v_e_n$v$ at the four corners of the u-cell[i, j-1], [i+1, j-1], [i, j], [i+1, j]
H_w_*, H_e_* (and likewise for $\eta$)cell-centred fields on a $2 \times 3$ patch (W/E in $x$, S/C/N in $y$)[i, *], [i+1, *]
β_w_c, β_e_ccell-centred basal friction at the two flanking cells[i, j], [i+1, j]
s_w_c, s_e_ccell-centred surface elevation at the two flanking cells[i, j], [i+1, j]

The y-momentum equation uses the mirror naming (v_0_m1, v_0_0, v_0_p1 in the cross direction; cell patch $3 \times 2$; etc.). See derive_y_residual for the full table.

Discrete strong-form residual (x-momentum)

using IceSheetStencils, Symbolics, Latexify
spec = derive_x_residual()
# Render the latexify output as a Documenter-style math container so KaTeX
# picks it up; raw \begin{equation} blocks would otherwise be printed
# verbatim and Markdown wrappers get HTML-escaped.
showmath(x) = HTML("<p class=\"math-container\">\\[" *
                   string(latexify(x; env = :raw)) * "\\]</p>")

Putting the membrane, cross-shear, drag and driving-stress terms together gives the discrete residual at a u-point:

showmath(spec.residual)

\[\frac{\frac{1}{4} \left( \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_n} \mathtt{\eta\_e\_n} + \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_n} \mathtt{\eta\_w\_n} \right) \left( \frac{ - \mathtt{u\_0\_0} + \mathtt{u\_0\_p1}}{\mathtt{dy}} + \frac{\mathtt{v\_e\_n} - \mathtt{v\_w\_n}}{\mathtt{dx}} \right) - \frac{1}{4} \left( \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_s} \mathtt{\eta\_e\_s} + \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_s} \mathtt{\eta\_w\_s} \right) \left( \frac{\mathtt{u\_0\_0} - \mathtt{u\_0\_m1}}{\mathtt{dy}} + \frac{\mathtt{v\_e\_s} - \mathtt{v\_w\_s}}{\mathtt{dx}} \right)}{\mathtt{dy}} + \frac{2 \mathtt{H\_e\_c} \left( \frac{\mathtt{v\_e\_n} - \mathtt{v\_e\_s}}{\mathtt{dy}} + \frac{2 \left( - \mathtt{u\_0\_0} + \mathtt{u\_p1\_0} \right)}{\mathtt{dx}} \right) \mathtt{\eta\_e\_c} - 2 \mathtt{H\_w\_c} \left( \frac{\mathtt{v\_w\_n} - \mathtt{v\_w\_s}}{\mathtt{dy}} + \frac{2 \left( \mathtt{u\_0\_0} - \mathtt{u\_m1\_0} \right)}{\mathtt{dx}} \right) \mathtt{\eta\_w\_c}}{\mathtt{dx}} + \frac{ - \frac{1}{2} \left( \mathtt{H\_e\_c} + \mathtt{H\_w\_c} \right) g \left( \mathtt{s\_e\_c} - \mathtt{s\_w\_c} \right) \rho}{\mathtt{dx}} - \frac{1}{2} \mathtt{u\_0\_0} \left( \mathtt{\beta\_e\_c} + \mathtt{\beta\_w\_c} \right)\]

Setting the residual to zero recovers $A\,x = b$. The 9 entries of A on this row are the partial derivatives of the residual w.r.t. the 9 velocity unknowns; the RHS is $-\mathrm{residual}\bigl|_{u = v = 0}$. Both are auto-generated by compile_stencil.

Stencil coefficients (x-momentum)

Each entry of the row of A produced by an x-momentum equation is a small expression in the local field values. Below they are extracted via Symbolics.jacobian and lightly simplified.

J = Symbolics.jacobian([spec.residual], spec.unknowns);

Membrane (longitudinal) couplings

The two u_m1_0 / u_p1_0 couplings are pure membrane stress. The coefficient is the same on both sides — exactly what we expect for a self-adjoint discretisation.

J[1, findfirst(==(:u_m1_0), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{4 \mathtt{H\_w\_c} \mathtt{\eta\_w\_c}}{\mathtt{dx}^{2}}\]

J[1, findfirst(==(:u_p1_0), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{4 \mathtt{H\_e\_c} \mathtt{\eta\_e\_c}}{\mathtt{dx}^{2}}\]

Cross-shear couplings

The u_0_m1 / u_0_p1 couplings come from the cross-shear stress. The factor of $1/4$ is the corner-average of $\eta H$ over the four cells surrounding the corner.

J[1, findfirst(==(:u_0_m1), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{\frac{1}{4} \left( \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_s} \mathtt{\eta\_e\_s} + \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_s} \mathtt{\eta\_w\_s} \right)}{\mathtt{dy}^{2}}\]

J[1, findfirst(==(:u_0_p1), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{\frac{1}{4} \left( \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_n} \mathtt{\eta\_e\_n} + \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_n} \mathtt{\eta\_w\_n} \right)}{\mathtt{dy}^{2}}\]

Diagonal entry

The diagonal collects every term where the central unknown u_0_0 appears: both membrane derivatives, both cross-shear corners, and the basal-drag interpolation $\tfrac12(\beta_{w,c} + \beta_{e,c})$.

J[1, findfirst(==(:u_0_0), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{ - \frac{1}{2} \mathtt{dx}^{2} \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} - 4 \mathtt{dy}^{2} \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} - \frac{1}{4} \mathtt{dx}^{2} \mathtt{H\_e\_n} \mathtt{\eta\_e\_n} - \frac{1}{4} \mathtt{dx}^{2} \mathtt{H\_e\_s} \mathtt{\eta\_e\_s} - \frac{1}{2} \mathtt{dx}^{2} \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} - 4 \mathtt{dy}^{2} \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} - \frac{1}{4} \mathtt{dx}^{2} \mathtt{H\_w\_n} \mathtt{\eta\_w\_n} - \frac{1}{4} \mathtt{dx}^{2} \mathtt{H\_w\_s} \mathtt{\eta\_w\_s} - \frac{1}{2} \mathtt{dy}^{2} \mathtt{dx}^{2} \mathtt{\beta\_e\_c} - \frac{1}{2} \mathtt{dy}^{2} \mathtt{dx}^{2} \mathtt{\beta\_w\_c}}{\mathtt{dy}^{2} \mathtt{dx}^{2}}\]

$u$–$v$ corner couplings

The four v_*_* couplings each combine two contributions: the corner-averaged cross-shear (with factor $1/4$) and the longitudinal stress at the cell adjacent to the corner (which contributes a factor $2$). The simplifier therefore lumps the local $\eta H$ with coefficient $1 + 8 = 9$ while the other three corners carry coefficient $1$. The asymmetry is between corners, not between equations — pairing $\partial R_x / \partial v$ against $\partial R_y / \partial u$ for the same physical edge gives identical entries (this is exactly what the symmetry test in the test suite checks).

J[1, findfirst(==(:v_w_s), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{\mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_s} \mathtt{\eta\_e\_s} + 9 \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_s} \mathtt{\eta\_w\_s}}{4 \mathtt{dx} \mathtt{dy}}\]

J[1, findfirst(==(:v_e_s), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{ - 9 \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} - \mathtt{H\_e\_s} \mathtt{\eta\_e\_s} - \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} - \mathtt{H\_w\_s} \mathtt{\eta\_w\_s}}{4 \mathtt{dx} \mathtt{dy}}\]

J[1, findfirst(==(:v_w_n), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{ - \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} - \mathtt{H\_e\_n} \mathtt{\eta\_e\_n} - 9 \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} - \mathtt{H\_w\_n} \mathtt{\eta\_w\_n}}{4 \mathtt{dx} \mathtt{dy}}\]

J[1, findfirst(==(:v_e_n), Symbol.(spec.unknowns))] |> simplify |> showmath

\[\frac{9 \mathtt{H\_e\_c} \mathtt{\eta\_e\_c} + \mathtt{H\_e\_n} \mathtt{\eta\_e\_n} + \mathtt{H\_w\_c} \mathtt{\eta\_w\_c} + \mathtt{H\_w\_n} \mathtt{\eta\_w\_n}}{4 \mathtt{dx} \mathtt{dy}}\]

Driving stress (right-hand side)

The RHS at a u-point is the driving stress with $H$ and $\partial s / \partial x$ interpolated to the face:

rhs = -Symbolics.substitute(spec.residual,
                            Dict(uk => 0 for uk in spec.unknowns))
rhs |> simplify |> showmath

\[\frac{\frac{1}{2} \left( \mathtt{H\_e\_c} + \mathtt{H\_w\_c} \right) g \left( \mathtt{s\_e\_c} - \mathtt{s\_w\_c} \right) \rho}{\mathtt{dx}}\]

i.e. $\rho\, g \,\bar H \,(s_{e,c} - s_{w,c})/dx$ with $\bar H = \tfrac12(H_{w,c} + H_{e,c})$.

y-momentum

The y-momentum equation is the mirror image of the above under $x \leftrightarrow y$ and $u \leftrightarrow v$. Inspect it directly with derive_y_residual — the same showmath recipe shown here applies.

Sparsity

Each row of A carries 9 nonzeros (5-point cross in the same component plus 4 corner couplings to the other component). With $N = N_x\,N_y$ unknowns per component the operator has

\[\mathrm{nnz}(A) \;=\; 2N \times 9 \;=\; 18\,N\,.\]

using SparseArrays
H = fill(1.0e3, 8, 6); η = fill(1e15, 8, 6); β = fill(1e9, 8, 6); s = zero(H);
A, _ = assemble_ssa(SSAFields(H, η, β, s); dx = 1e3, dy = 1e3);
size(A), nnz(A), 18 * length(H)
((96, 96), 864, 864)

The matrix is symmetric to machine precision (the test suite checks $\|A - A^T\|\,/\,\|A\| \approx 2 \times 10^{-17}$) and negative-definite for strictly positive $\beta$, so it is solved directly with A \ b (Julia's sparse \ dispatches to UMFPACK / CHOLMOD).

For a worked variable-coefficient validation — a method-of-manufactured- solutions test that confirms second-order convergence — see test/runtests.jl.
